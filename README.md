## Requisitos

- nodejs v 12.x
- nodemon
- mongodb

## Instalacion

Para el Backend
```
  cd bckshateam
  npm install
```

Para el Frontend
```
  cd frtshateam
  npm install
```

## Iniciar Servicios Desarrollo

en la carpeta ```bckshateam``` correr ```nodemon```
